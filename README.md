# FrakkturResurgence
School showcase project focused on full-stack web development, combining frontend design with backend functionality. The goal was to build a functional e-commerce template from scratch, covering the complete workflow from UI/UX to deployment.

## Frontend development
- Built with React, TypeScript, Vite, and Tailwind CSS
- Component-based architecture with reusable UI elements
- Responsive layout optimized for desktop and mobile
- Clean, modern design focused on usability

## Core features
- User authentication (login and registration)
- Product listing and detail views
- Shopping cart and wishlist functionality
- Basic user account state management

## Backend & database
- MySQL database for persistent data storage
- Tables for users, products, carts, wishlists, and orders
- PHP-based backend endpoints for authentication and data handling
- Database administration and testing via phpMyAdmin

## Deployment & hosting
- Hosted on Endora shared hosting
- Production build uploaded using WebFTP
- Environment-based configuration for correct frontend routing
- Real-world shared hosting setup without cloud platforms

## Project purpose
- Demonstrate practical frontend and backend skills
- Show understanding of database-driven web applications
- Practice real deployment workflows used in small to medium projects

Overall, FrakkturResurgence demonstrates practical skills in frontend development, backend integration, database design, and real world deployment on shared hosting mirroring how small to mid scale web projects are often built and maintained outside of purely local or cloud native environments.
